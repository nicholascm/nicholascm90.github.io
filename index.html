Pomodoro Clock

The Pomodoro Clock allows the user to set a work session and break session length. Once the timer is started, sessions alternate while keeping the user aware of their progress. The project utilizes Bootstrap and jQuery. Separately from the challenge, I also threw in a To Do list from the AngularJS.org tutorials that makes this application infinitely more useful for me at work.
